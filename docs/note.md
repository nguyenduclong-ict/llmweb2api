# Cải thiện hiệu năng deepseek khi chạy song song nhiều request trên cùng 1 tài khoản

Bối cảnh:

- 1 tài khoản deepseek
- Api key bật cache

mở 2 cửa sổ chat opencode dùng endpoint này:

- 1 của sổ chat hoạt động bình thường
- 1 của sổ chat thường xuyên bị stop response giữa chừng

Yêu cầu:

- Điều tra nguyên nhân
- Đề xuất hướng giải quyết
